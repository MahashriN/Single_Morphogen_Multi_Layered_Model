# Single Morphogen Multi-Layered Model

This repository contains MATLAB code used to generate numerical results for a research study on pattern formation in coupled multilayer reaction–diffusion systems.

The simulations are implemented using a finite element method in two spatial dimensions, with support for multiple coupled layers, nonlinear reaction kinetics, and nonlinear boundary flux terms.

### How to run the code 
1. Generate the mesh  
   - `meshRectangle2D.m` - This script generates structured rectangular meshes for layered domains.

   The number of layers is controlled by the parameter N inside the script:
     - N = 1 : single layer
     - N = 2 : two layers
     - N = 3 : three layers
   
   The domain length L, layer height H, and mesh spacing dx can be modified directly in this file.

   If save_file = 1, the script saves the mesh data as .mat files, including:
     - `p1_2D.mat`, `t1_2D.mat`
     - `p2_2D.mat`, `t2_2D.mat`
     - `p3_2D.mat`, `t3_2D.mat`
       
   along with corresponding top and bottom boundary node and element files for each layer (e.g. `bottom_p1_1D.mat`, `top_t2_1D.mat`, etc.).

   These files contain the nodes and elements for the full 2D domain and the boundary edges required for boundary flux calculations.

   The generated mesh files are saved in folders labelled by the geometry parameters N, L, and H.
   
   For example, the folder

   - N=2/L=20/H=0.1/

   contains the mesh data for a two-layer domain with length L = 20 and layer height H = 0.1, including the corresponding p1, p2 (and p3 where applicable) node and element files.

3. Run the main simulation  
   - `Main_2L_y2D.m` - Main script for simulations of two coupled layers in 2D.
   - `Main_3L_y2D.m` - Main script for simulations of three coupled layers in 2D.
   
   These scripts:
   - load the mesh data generated by meshRectangle2D.m
   - define model parameters and initial conditions
   - assemble the finite element matrices
   - compute reaction kinetics and boundary fluxes
   - advance the solution in time

### Main functions used  
The main scripts call the following core routines:
  - `AssembleGlobalMatrices2D.m` - Assembles global mass and stiffness matrices for the 2D finite element problem.
  - `ReactKineInt.m` - Computes integrals associated with nonlinear reaction kinetics.
  - `NLBoundFluxInt.m` - Computes nonlinear boundary flux integrals.

These routines rely on the following helper files: `basis_linear_1D.m`, `basis_linear_2D.m`, `RefElemQuad.m`, `RefEdgeQuad.m` 
which define linear basis functions and reference quadrature elements in 1D and 2D.

### Notes
- All parameters controlling geometry, coupling, and kinetics are defined inside the main scripts.
- The code is written for clarity and reproducibility rather than computational optimisation.
- The repository is intended to accompany a research paper and reproduce its numerical results.
